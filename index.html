📖 하루 운영 가이드
IBD/오닐·미너비니 방법론 기반 · 한국시간 기준 (미국 서머타임)
🌅 Phase 1 · 환경 파악 아침 7~9시 (장 마감 직후)
목표: "오늘 얼마나 공격적이어야 하는가?" · 소요 5~10분
  1. Market Pulse 탭 먼저 — Regime 확인 (강한 상승추세 / 상승추세 / 중립 / 조정압력 / 하락방어)
  2. 권장 투자 비중 숫자 확인 → 오늘의 최대 신규 진입 예산 결정
  3. 시장 폭(Breadth) 게이지 점검 — Stage2 비율, 50일선 위 비율이 높을수록 공격적으로
  4. 조정압력·하락방어면 오늘은 신규 진입 자제, 보유 종목 관리에 집중
🔍 Phase 2 · 섹터·종목 발굴 아침 (스캔 직후)
목표: "오늘 사거나 지켜볼 3~7개 종목 확정" · 소요 20~40분
  1. 섹터 분석 탭 — 돈이 어디로 흐르는지 확인. 강세 섹터(초록)부터 종목을 찾는다 ("시장 → 섹터 → 종목" 순서)
  2. 🎯 Confluence 탭 — 여러 시그널이 겹친 종목 상위부터 확인. 점수 높을수록 여러 각도에서 강세 확인된 종목
  3. 스크리너 탭 — RS 80+ · Stage 2 · 진입 후보 위주로 좁히기
  4. 피벗 거리 0 ~ +2%인 종목 = 매수 타이밍 임박. +10% 넘으면 추격 금지 (베이스 매수 윈도우 만료)
💰 Phase 3 · 매매 실행 예약주문 → 밤 정규장 체결
진입 규칙 (O'Neil Pivot Rule)
  • 피벗 돌파 당일 진입 (종가가 피벗 위 유지)
  • 거래량 50일 평균 +40% 이상 필수 — 없으면 가짜 돌파(Fake Out) 가능성
  • 피벗 위 +5% 이상 추격 금지 (Extended)
  • 50% → 25% → 25% 분할 매수 (미너비니 피라미딩)
  • 손절선 −7~8% (시장이 약하면 −5~6%로 타이트하게)
  • 한국에선: 아침에 지정가 예약주문 → 밤 정규장에서 자동 체결
보유 종목 관리 (전략실 탭)
  • ⚠ 탈출 신호(빨간 마크) 뜬 보유 종목 = 즉시 청산 검토
  • Stage 4 하향 전환 / RS 약화 = 익절·청산
  • 정체된 종목은 종목 스위칭으로 더 강한 종목 교체 검토
📝 Phase 4 · 복기·기록 스캔 후 (저녁/다음 아침)
목표: 오늘의 변화 점검 + 내일 준비
  1. 2_스캔실행.bat 실행 → 다음 날 데이터로 갱신 (수익률·탈출 신호 자동 재계산)
  2. NEW 배지 종목 = 오늘 새로 뜬 후보 → 내일 진입 검토
  3. CON 배지 종목 = 시그널 겹친 강세 → 우선순위
  4. 보유 종목이 Phase 6/7(후반·분배)로 떨어졌으면 다음 날 매도 계획
  5. 날짜 드롭다운으로 과거 기록 확인 가능 — 며칠 전 후보가 어떻게 됐는지 복기
⚖️ 핵심 원칙 (O'Neil · Minervini)
  • 시장 먼저 — 시장이 하락 추세면 아무리 좋은 종목도 사지 않는다 (M = Market Direction)
  • 추격 금지 — 피벗에서 멀어진(+10%↑) 종목은 다음 기회를 기다린다
  • 손절은 기계적으로 — −7~8% 도달 시 감정 없이 매도 (큰 손실 방지가 최우선)
  • 이익은 길게 — Lock Ratchet으로 수익을 단계적으로 보호하며 추세를 끝까지
  • 분산보다 집중 — 동시 보유 ~12개 한도. 가장 강한 종목에 집중
